## Criterio de uso

- Comparar estructura, composición, densidad, luz, color y movimiento; no copiar literalmente una obra.
- Mantener estos originales sin recortar ni recomprimir. Las anotaciones y comparativas deben guardarse como archivos nuevos.
- Documentar aquí cualquier referencia nueva antes de usarla para calibrar un proyecto.
